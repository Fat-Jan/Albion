## 欧服可用接口

### 1. Gameinfo API

欧服根地址：

```text
https://gameinfo-ams.albiononline.com/api/gameinfo
```

社区文档来源：Tools4Albion API info 明确列出三个 Gameinfo 根地址：

- Americas：`https://gameinfo.albiononline.com/api/gameinfo/`
- Europe：`https://gameinfo-ams.albiononline.com/api/gameinfo/`
- Asia：`https://gameinfo-sgp.albiononline.com/api/gameinfo/`

路径与亚服一致，常用端点：

```text
/search?q=<name>
/players/<ID>
/players/<ID>/kills
/players/<ID>/deaths
/players/statistics
/guilds/<ID>
/guilds/<ID>/members
/guilds/<ID>/data
/battles
/battles/<ID>
/events
/events/<ID>
/events/playerfame
/events/guildfame
/events/killfame
/events/playerweaponfame
```

本机实测：

```bash
curl 'https://gameinfo-ams.albiononline.com/api/gameinfo/search?q=Top%20Squad'
curl 'https://gameinfo-ams.albiononline.com/api/gameinfo/events?limit=2&offset=0'
```

结果：均返回 JSON。`/events` 返回最新欧服击杀事件，字段包含 `EventId`、`TimeStamp`、`Killer`、`Victim`、`Equipment`、`Inventory` 等。

### 2. AODP 市场 API

欧服根地址：

```text
https://europe.albion-online-data.com
```

AODP 官方 API 文档明确列出 Europe host。路径与亚服一致：

```text
/api/v2/stats/prices/<item_ids>.json
/api/v2/stats/history/<item_ids>.json
/api/v2/stats/gold.json
```

本机实测：

```bash
curl 'https://europe.albion-online-data.com/api/v2/stats/prices/T4_BAG.json?locations=Caerleon&qualities=1'
```

结果：返回欧服市场 JSON，示例字段包含：

```text
item_id, city, quality, sell_price_min, sell_price_min_date,
sell_price_max, buy_price_min, buy_price_max
```

注意：AODP 官方文档写明限流为 180 requests / 1 minute、300 requests / 5 minutes，URL 长度上限 4096。

### 3. AlbionBB 战报 API

欧服前端：

```text
https://europe.albionbb.com
```

欧服 API region 实测不是 `europe`，而是 `eu`：

```text
https://api.albionbb.com/eu
```

本机实测：

```bash
curl 'https://api.albionbb.com/eu/battles?page=0&minPlayers=20'
```

结果：返回欧服战役列表 JSON。

以下路径返回 404，不能用于欧服：

```text
https://api.albionbb.com/europe/battles
https://api.albionbb.com/ams/battles
https://api.albionbb.com/west/battles
```

常用端点：

```text
https://api.albionbb.com/eu/battles?page=0&minPlayers=20
https://api.albionbb.com/eu/battles/kills?ids=<battle_id>
```

## 当前 bot 欧服配置

当前仓库默认环境变量：

```env
GAMEINFO_BASE=https://gameinfo-ams.albiononline.com/api/gameinfo
AODP_BASE=https://europe.albion-online-data.com
ALBIONBB_BASE=https://api.albionbb.com/eu
ALBIONBB_WEB_BASE=https://europe.albionbb.com
KILLBOARD_SERVER=live_ams
```

当前代码链接已改为可配置：

- `bot/cards/query_cards.py` 的官方击杀板链接使用 `config.KILLBOARD_SERVER`，默认 `live_ams`。
- `bot/albion/battle_report.py` 的 AlbionBB 战报链接使用 `config.ALBIONBB_WEB_BASE`，默认 `https://europe.albionbb.com`。
- `README.md`、`使用说明书.md`、`阿尔比恩数据接口文档.md` 的默认标识已同步为欧服；亚服字符串只保留在历史说明、其他区说明或手动回切示例里。
